### Amazon Framework — How it works
- Scrapes Amazon new-releases + best-sellers across 15 category URLs × 2 pages = ~900 unique books/cycle
- Uses Web Unlocker (not browser) — browser only returns 2 books, Web Unlocker returns 30-50
- HTML parser extracts title (p13n-sc-css-line-clamp class, 1st match) + author (2nd match)
- For each author: fires 14 sources simultaneously (Google×3, Bing×2, DDG, Yahoo, Instagram, Reedsy, Goodreads, Amazon author page, press search)
- Email validation: `isLikelyAuthorEmail()` — requires author name in email local or domain, fuzzy match for spelling variants
- 30 authors processed concurrently
- Loops back after all categories done, waits 10 min for new listings
